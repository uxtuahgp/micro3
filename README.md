## Домашнее задание по теме Микросервисы - подходы ##  

### Задача 1: Обеспечить разрабтку ###  
Многие продукты на сегодня предлагают как облачное, так и On-Premises развертывание.  
Можно рассмотреть как MS Azure DevOps, так и GitLab, коммерческие версии которых предоставляют примерно сходный набор инструментов.  
Наиболее широко распространным решением, отвечающим требованиям является GitLab.  
- GitLab имеет бесплатную версию с ограничениями в возможностях настройки CI/CD и платную полнофункциональную версию.  
- Из самого названия следует, что в основе лежит Git.  
- В продукте имеется возможность хранения множества проектов.  
- Система предоставляетразвитый инструментарий для построения CI/CD с помощью собственных агентов сборки runner-ов.  
- Система позволяет параллелить как сборку, так и тестирование.  
- Запуск сборк возможен как по событиям, так и по кнопке с передачей параметров.  
- Храненеи секретов можно организовать как снаружи, например в Vault или Openbao, так и внутри с помощью GitLab Secrets Manager.
